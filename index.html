Tips & Examples

Function Examples:

• x^2 (quadratic)

• sin(x) (sine wave)

• cos(x) (cosine wave)

• tan(x) (tangent)

• csc(x) (cosecant)

• sec(x) (secant)

• cot(x) (cotangent)

• log(x) (logarithm)

• sqrt(x) (square root)

• abs(x) (absolute value)

Syntax Rules:

• Use * for multiplication

• Use ^ for powers

• Use pi for π

• Use e for Euler's number

• Use parentheses for grouping

Complex Examples:

• x^3 - 4*x^2 + x - 1

• sin(x) + cos(x)

• 2*x + 3

• sqrt(x^2 + 1)

• 1/(x^2 + 1)
